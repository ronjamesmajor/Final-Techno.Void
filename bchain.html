Cardano is a blockchain project which utilizes the ADA Token, named after Ada Lovelace, who is regarded as the first “computer programmer”. But what’s more important is, when is it ever going to break $3.00 again, because I’m down like 10% still...


Cardano was founded and launched in September 2017, after a two year period of token distribution, by Ethereum co-founder Charles Hoskinson, and is a BlockChain 3.0 Project. The entire idea of Cardano is to build off of what was created by Bitcoin and Ethereum while using a first of it’s kind, peer-reviewed research based, proof-of-stake model to deliver a scalable and energy-efficient Smart Contract platform using the Ouroboros consensus mechanism. Cardano utilizes the Plutus Platform which is based upon Haskell. Cardano also provides a sandboxing environment called the Marlowe Playground, a domain specific language based on Haskell, where users can practice writing and test smart contracts, specifically for Decentralized Finance (DeFi).

Cardano was founded to take the basis of what Ethereum was and grow it to deliver scalable smart contracts and also increase energy efficiency by using a proof-of-stake model as opposed to the Blockchain 1.0 and 2.0 proof-of-work model.

As Cardano uses a decentralized network it is currently believed that Cardano is unhackable. However, it has been suggested by Cardanians.IO on Medium that because of the “no single point of failure” system Cardano employs, and because of it’s enhanced security, that even if an intruder penetrated into a Cardano node, the only thing they could access is a Pool Operator’s node and pool operators do not store ADA coins of Delegators. Delegators keep coins in wallets, and delegate coins without sending them physically to anyone. Therefore even if a hacker penetrates into the pool, consensus power will be kept in the hands of delegators. In order to damage the consensus directly hackers would be required to hack into a massive number of nodes simultaneously which is nearly impossible, especially considering each node is ideally operated independently so security measures are different per node, so the same method of attack cannot be used on all nodes. Even if in the unlikely event this could happen, new blocks are appended to the Cardano chain every 20 seconds, so any slowdown would be minimal and unlikely to be noticed. This makes Cardano effectively immutable. However, like all software it is not without bugs, and as of this writing, 11 bugs have been reported to Cardano through it’s bug bounty program which pays white-hat hackers to find flaws in code.

Because of Cardano’s versatility, it can provide a host of opportunities for software development on the platform, including wallet integration, native tokens, NFT’s, smart contracts, staking pool operations, building transaction metadata, and more. It also provides to it’s userbase opportunities to engage with existing traditional economic infrastructure in a way that is both visionary and cutting edge. Companies and individuals can use Cardano for the issuance and verification of credentials like diplomas and ID badges, it can prevent counter fitting it can assist in supply chain tracking and also, digital entity and ID verification and issuance. In August of 2021 Cardano also rolled out a capability upgrade called Alonzo which enables the development of Dapps and DeFi smart contracts.

Cardano also has much of the benefit and security of Ethereum with a more environmentally approachable blueprint, and is based on peer reviewed consensus algorithms. This makes Cardano appear to be something that could be poised for large successes as the world moves toward more smart contracts and more blockchain technologies.
